# @eove/eslint-config-node

Node eslint shareable configuration

## Installation

```
npm install --save-dev eslint@8 eslint-plugin-prettier@4 @eove/eslint-config-node
```

## Usage

Create a `.eslintrc.js` file with:

```js
module.exports = {
  root: true,
  extends: ["@eove/eslint-config-node"],
};
```
